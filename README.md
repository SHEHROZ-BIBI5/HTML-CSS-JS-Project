 HTML-CSS-JS-Project

A collection of **HTML, CSS, and JavaScript projects** for learning and practicing web development.  
This repository includes small projects, games, and basic web pages to improve frontend skills.

## Projects Included

- **Calculator** – Interactive calculator using HTML, CSS, and JavaScript  
- **Login Page** – Simple login form with HTML and CSS styling  
- **Rock Paper Scissors Game** – Fun interactive game using JavaScript  
- **Simple Website** – Basic website structure with CSS and JavaScript  
- **Contact Page** – HTML form layout with formatting

## Tech Stack

- **HTML5** – Structure and markup  
- **CSS3** – Styling, layout, and responsive design  
- **JavaScript (ES6)** – Interactivity, DOM manipulation, and logic
